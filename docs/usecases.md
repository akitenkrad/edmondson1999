# Usecases

## 1. Baseline reproduction

```bash
cargo run --release -- run \
    --n-teams 90 --team-size 8 \
    --lambda 0.10 --alpha 0.30 --beta 0.25 --gamma 0.50 --delta 0.35 \
    --t-max 24 --runs 30 --seed 1999
uv run edmondson-tools visualize --results-dir results/latest
uv run edmondson-tools reproduce  --results-dir results/latest
```

The `run` prints each trial's ICC(ψ), ψ→L slope/R², and mediation ratio, then a pooled summary; `reproduce` checks every §5 anchor band.

## 2. Sensitivity analysis (α × δ)

```bash
cargo run --release -- sweep \
    --alpha-min 0.10 --alpha-max 0.50 --alpha-step 0.05 \
    --delta-min 0.10 --delta-max 0.60 --delta-step 0.10 \
    --runs 30 --seed 1999
uv run edmondson-tools visualize-sweep --results-dir results/latest
```

Sweeping the context-support weight α and the shared-belief convergence weight δ shows how ICC(ψ), the mediation ratio, and the ψ→L R² move across the parameter grid. δ controls how strongly within-team beliefs converge (and so the between-team share of ψ variance, i.e. ICC); α controls how strongly support pulls ψ up.

## 3. LLM-driven ablation

```bash
export OLLAMA_HOST=http://localhost:11434
export OLLAMA_MODEL=llama3.1
cargo run --release -- run --decision-mode llm \
    --cache-path runs/edmondson_cache.json \
    --n-teams 90 --runs 10 --seed 1999
```

Swaps the rule-based VOICE logit for an LLM that decides VOICE/SILENCE and the learning behavior from the individual's inner state and team context. Re-running with the same arguments replays from the warm cache (≈100% cache hits, near-zero new calls). The deterministic socsim core (init, networks, the five non-LLM mechanisms) is unchanged; only the decision layer differs.

## 4. Network family

`--network-model {watts-strogatz|erdos-renyi|barabasi-albert}` changes the within-team topology. Watts–Strogatz with a mid-range `--network-beta` (≈0.15) is the design default; the others are sensitivity checks.
